Mindcraft is not embedded on the public site

Mindcraft is a separate Node.js app that listens on http://127.0.0.1:43110/. This hub is served over HTTPS, so the browser cannot load that local HTTP URL inside the frame (mixed content + your machine only).

To use Mindcraft: run it on your computer, then open the launcher in a new tab from the main hub or visit the URL above while the server is running.

From your game projects folder, typical start:

cd "Kimi k2.6 mindcraft/Kimi k2.6 mindcraft" npm install npm start

For the full hub with embed + auto-start of the launcher, run Website/serve.sh on localhost.
